## CardTabs

與 [Ant Design](https://ant.design/) 的 [Tabs](https://4x.ant.design/components/tabs/) 組件類似，但 `CardTabs` 采用了卡片樣式和邊框。

### 導入組件

```jsx | pure
import {CardTabs} from '@yookue/ant-buddy-pro';
```
