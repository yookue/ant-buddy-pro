## RefreshImage

RefreshImage 是一個可點擊刷新的圖片組件。

典型的場景是配合圖形驗證碼使用，尤其是服務端不使用 Node.js 而生成圖形驗證碼的情況。

### 導入組件

```jsx | pure
import {RefreshImage} from '@yookue/ant-buddy-pro';
```
