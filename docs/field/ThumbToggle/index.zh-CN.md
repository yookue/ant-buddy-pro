## ThumbToggle

ThumbToggle 用于显示一个拇指图标，并可以切换。

典型的应用场景是，显示两个拇指，分别为 `喜欢` 和 `不喜欢`。

### 导入组件

```jsx | pure
import {ThumbToggle} from '@yookue/ant-buddy-pro';
```
