## ThumbTuple

ThumbTuple 用於顯示兩個拇指圖標和計數。

典型的應用場景是，顯示兩個拇指，分別為 `喜歡` 和 `不喜歡`。

### 導入組件

```jsx | pure
import {ThumbTuple} from '@yookue/ant-buddy-pro';
```
