## ThumbTuple

ThumbTuple 用于显示两个拇指图标和计数。

典型的应用场景是，显示两个拇指，分别为 `喜欢` 和 `不喜欢`。

### 导入组件

```jsx | pure
import {ThumbTuple} from '@yookue/ant-buddy-pro';
```
