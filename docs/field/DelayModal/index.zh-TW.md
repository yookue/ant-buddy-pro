## DelayModal

DelayModal 用於在超時後顯示一個模態對話框，如果沒有其它事件來阻止（延遲）它的話。

典型的應用場景是，當用戶長時間不操作後，顯示對話框來提示用戶需要重新登錄。

### 導入組件

```jsx | pure
import {DelayModal} from '@yookue/ant-buddy-pro';
```
