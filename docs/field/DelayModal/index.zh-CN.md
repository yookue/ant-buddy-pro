## DelayModal

DelayModal 用于在超时后显示一个模态对话框，如果没有其它事件来阻止（延迟）它的话。

典型的应用场景是，当用户长时间不操作后，显示对话框来提示用户需要重新登录。

### 导入组件

```jsx | pure
import {DelayModal} from '@yookue/ant-buddy-pro';
```
