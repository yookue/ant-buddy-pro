## NotifyBadge

NotifyBadge，提供了一個帶下拉框的徽標。

典型的應用場景是，顯示一個小圖標，下拉展示通知和任務列表。

### 導入組件

```jsx | pure
import {NotifyBadge} from '@yookue/ant-buddy-pro';
```
