## CompactTuple

CompactTuple 可以显示紧凑间距模式下的一个控件和一个附加节点。

典型的应用场景是，显示一个证件类型的日期选取框和一个切换/复选框，用于标记证件是长期有效。

在这种情况下，日期选取框是有边框的，但切换/复选框没有。此控件可以给切换/复选框加上边框，以便统一用户 UI。

### 导入组件

```jsx | pure
import {CompactTuple} from '@yookue/ant-buddy-pro';
```
