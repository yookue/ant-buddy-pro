## ChronoTuple

ChronoTuple, 提供了一个可以输入数值和选择时间单位的选择框，选项数据来源于 [ChronoUnit](https://docs.oracle.com/javase/8/docs/api/java/time/temporal/ChronoUnit.html)。

### 导入组件

```jsx | pure
import {ChronoTuple} from '@yookue/ant-buddy-pro';
```
