## ChronoTuple

ChronoTuple, 提供了一個可以輸入數值和選擇時間單位的選擇框，選項數據來源於 [ChronoUnit](https://docs.oracle.com/javase/8/docs/api/java/time/temporal/ChronoUnit.html)。

### 導入組件

```jsx | pure
import {ChronoTuple} from '@yookue/ant-buddy-pro';
```
