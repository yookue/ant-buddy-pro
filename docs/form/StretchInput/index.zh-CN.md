## StretchInput

StretchInput，提供了一个可以动态拉伸的文本框。

典型的应用场景是，显示一个小的搜索框或搜索图标，当点击它时拉伸显示它。

### 导入组件

```jsx | pure
import {StretchInput} from '@yookue/ant-buddy-pro';
```
