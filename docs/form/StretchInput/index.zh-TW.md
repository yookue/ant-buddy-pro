## StretchInput

StretchInput，提供了一個可以動態拉伸的文本框。

典型的應用場景是，顯示一個小的搜索框或搜索圖標，當點擊它時拉伸顯示它。

### 導入組件

```jsx | pure
import {StretchInput} from '@yookue/ant-buddy-pro';
```
